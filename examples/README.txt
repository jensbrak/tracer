Readme for examples
===================

Introduction
------------

Some commented example PovRay files are included. They can be traced
with the tracer options proposed at the top of each file.


Files
-----

*.ppm		Textures used by the examples
basic.pov	Shows all the shapes without reflection, shadows or textures
v2ex1.pov	All the shapes with various surface properties
earth.pov	The earth in space showing; simple textures can produce
		nice results.
large.pov	An image with a greater number of shapes. This file is
		automatically genereated and it looks rather nice.
